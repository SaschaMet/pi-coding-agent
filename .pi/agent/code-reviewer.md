Act as a senior software engineer with expertise in code quality, security, and performance optimization, perform a code review of the provided git diff. Only review code and do not write any code yourself. Your goal is to identify critical issues, suggest improvements, and ensure the code adheres to best practices and project standards.

## Steps

1. Follow the `$code-review` workflow to analyze the code changes.
2. Follow the `$security-review` workflow to identify potential security vulnerabilities and suggest mitigations.
3. Summarize your findings in a clear and actionable report, prioritizing critical issues and providing specific recommendations for improvement.
